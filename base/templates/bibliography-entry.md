# Vorlagen für Literaturverzeichnis-Einträge

> Diese Vorlagen zeigen das korrekte Format für verschiedene Quellentypen.
> Das tatsächliche Format hängt vom konfigurierten Zitationsstil ab.

## Harvard Inline

### Buch
```
Nachname, V. (Jahr): Titel. Auflage. Verlag, Ort.
```
Beispiel: `Müller, M. (2024): Digitale Transformation im Mittelstand. 2. Aufl. Springer, Berlin.`

### Journal-Artikel
```
Nachname, V./Nachname, V. (Jahr): Titel. In: Journalname, Jg. X, Nr. Y, S. X-Y.
```
Beispiel: `Schmidt, K./Weber, L. (2023): KI-Adoption in KMU. In: Wirtschaftsinformatik, Jg. 65, Nr. 3, S. 234-251.`

### Sammelband
```
Nachname, V. (Jahr): Kapiteltitel. In: Herausgeber, V. (Hrsg.): Sammelbandtitel. Verlag, Ort, S. X-Y.
```

### Internetquelle
```
Organisation/Autor (Jahr): Titel. URL: [URL], Abruf: TT.MM.JJJJ.
```
Beispiel: `KPMG (2024): KI-Studie Deutschland. URL: https://..., Abruf: 15.01.2026.`

### Konferenzbeitrag
```
Nachname, V. (Jahr): Titel. In: Konferenzname, Ort, S. X-Y.
```

## APA 7

### Buch
```
Nachname, V. (Jahr). Titel (X. Aufl.). Verlag.
```

### Journal-Artikel
```
Nachname, V. & Nachname, V. (Jahr). Titel. Zeitschrift, Band(Heft), Seiten. https://doi.org/xxx
```

## IEEE

### Buch
```
[1] V. Nachname, Titel, X. Aufl. Ort: Verlag, Jahr.
```

### Journal-Artikel
```
[2] V. Nachname und V. Nachname, "Titel," Zeitschrift, Bd. X, Nr. Y, S. X-Y, Jahr.
```

## Chicago (Author-Date)

### Buch
```
Nachname, Vorname. Jahr. Titel. Auflage. Ort: Verlag.
```

### Journal-Artikel
```
Nachname, Vorname. Jahr. "Titel." Zeitschrift Band, Nr. Heft: Seiten.
```
