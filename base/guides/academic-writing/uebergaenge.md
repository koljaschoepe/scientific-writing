# Übergänge zwischen Kapiteln

> Diese Datei definiert die ZENTRALEN Regeln für Kapitelübergänge.
> Alle Schreib- und Review-Agenten müssen diese Regeln einhalten.

## Kernprinzip: Implizite Verbindung

Der rote Faden entsteht durch inhaltliche Verknüpfung über Konzeptnamen,
NICHT durch explizite Ankündigungen oder Zusammenfassungen.

## VERBOTEN am Kapitelende

Der letzte Satz eines Kapitels darf NICHT:
- Das nächste Kapitel ankündigen
- Vorwegnehmen was erst später kommt
- Explizit auf ein anderes Kapitel verweisen

```
NEIN: "...wird im folgenden Abschnitt dargelegt."
NEIN: "Das nächste Kapitel behandelt..."
NEIN: "Im Folgenden wird erläutert, wie..."
NEIN: "Hierzu wird in Kapitel 3 näher eingegangen."
```

STATTDESSEN: Mit einer inhaltlichen Schlussfolgerung oder einem offenen
Gedanken enden, der die Notwendigkeit des nächsten Themas impliziert.

```
JA: "Es bedarf eines ganzheitlichen Ansatzes, der die Prinzipien
    der Spezifikation und Modularisierung auf den Prozess der
    KI-gestützten Textproduktion überträgt."
    -> Das nächste Kapitel greift dies auf, ohne Ankündigung.
```

## VERBOTEN am Kapitelanfang

Der erste Satz eines Kapitels darf NICHT:
- Das vorherige Kapitel zusammenfassen
- Den letzten Satz des Vorgängers paraphrasieren
- Explizit auf das vorherige Kapitel verweisen

```
NEIN: "Wie im vorangegangenen Abschnitt dargelegt wurde..."
NEIN: "Aufbauend auf den in Kapitel 2 erläuterten Grundlagen..."
NEIN: "Nachdem die theoretischen Grundlagen dargelegt wurden..."
NEIN: "Die in Kapitel 2.2 dargestellten Konzepte zeigen..."
```

STATTDESSEN: Direkt mit inhaltlich Neuem beginnen. Die Verbindung
zum Vorgänger ergibt sich durch die Verwendung der gleichen Konzeptnamen.

```
JA: "Organizational Memory beschreibt die Fähigkeit einer Organisation,
    Wissen systematisch zu speichern und abzurufen."
    -> Greift ein Konzept auf das im vorherigen Kapitel eingeführt wurde,
    ohne es explizit als Rückverweis zu formulieren.
```

## Erlaubte Verbindungen

- Konzeptnamen wiederverwenden (implizite Verbindung)
- Auf die gleiche Forschungsfrage Bezug nehmen
- Fachbegriffe aus dem vorherigen Kapitel weiterführen
- Bei größeren thematischen Sprüngen (selten, nur zwischen Hauptkapiteln):
  "Auf Basis dieser Erkenntnisse lassen sich konkrete Anforderungen ableiten."

## Signalwörter für Kohärenz innerhalb eines Kapitels

```
Darüber hinaus...
Ferner...
Zudem...
Vor diesem Hintergrund...
Dies führt zu...
Folglich...
```

## Prüfkriterien (für Review-Agenten)

Für JEDES Kapitel (außer dem ersten):
1. Enthält der erste Satz inhaltlich Neues?
2. Paraphrasiert der erste Satz NICHT den letzten Satz des Vorgängers?
3. Gibt es KEINE expliziten Kapitelverweise?
4. Kündigt der letzte Satz NICHT das nächste Kapitel an?
5. Ist die Verbindung zum Vorgänger durch Konzeptnamen erkennbar?
