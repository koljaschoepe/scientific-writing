# Formulierungshilfen

## Einleitung eines Themas
```
Die vorliegende Arbeit befasst sich mit...
Im Mittelpunkt der Untersuchung steht...
Gegenstand dieser Analyse ist...
Die zentrale Fragestellung lautet...
```

## Argumentation aufbauen
```
Vor diesem Hintergrund zeigt sich...
In diesem Zusammenhang ist festzustellen...
Daraus ergibt sich die Notwendigkeit...
Dies führt zu der Erkenntnis, dass...
Hieraus lässt sich ableiten, dass...
```

## Quellen einführen
```
Laut [Autor] (Jahr, S. X)...
[Autor] (Jahr) argumentiert, dass...
Nach Auffassung von [Autor]...
Wie [Autor] (Jahr, S. X) darlegt...
Einer Studie von [Autor] zufolge...
[Autor] definiert [Begriff] als...
```

## Gegenüberstellung
```
Während [A] argumentiert..., vertritt [B] die Position...
Im Gegensatz zu [A] betont [B]...
[A] und [B] stimmen darin überein, dass...
Anders als [A] sieht [B]...
```

## Kritische Würdigung
```
Diese Argumentation lässt sich kritisch hinterfragen...
Einschränkend ist anzumerken, dass...
Dieser Ansatz weist jedoch Limitationen auf...
Es bleibt offen, inwieweit...
Problematisch erscheint dabei...
```

## Schlussfolgerungen
```
Zusammenfassend lässt sich festhalten, dass...
Aus den dargelegten Erkenntnissen ergibt sich...
Die Analyse zeigt, dass...
Es wird deutlich, dass...
```

## Ergebnisse einordnen
```
Diese Ergebnisse deuten darauf hin, dass...
Bemerkenswert ist dabei, dass...

Dies steht im Einklang mit den Erkenntnissen von [Autor]...
```
