# Verbotene Muster

## Ich-Form

```
NEIN: Ich werde in dieser Arbeit untersuchen...
NEIN: Meiner Meinung nach...
NEIN: Ich bin der Ansicht, dass...

JA: Die vorliegende Arbeit untersucht...
JA: Es lässt sich argumentieren, dass...
JA: Aus der Analyse ergibt sich...
```

## Umgangssprache

```
NEIN: Das ist echt wichtig...
NEIN: Die Sache ist die...
NEIN: Im Grunde genommen...
NEIN: eigentlich, irgendwie, sozusagen, quasi
```

## Journalistische Übertreibungen

```
NEIN: enorm, phänomenal, gigantisch, sensationell, dramatisch
NEIN: revolutionär, bahnbrechend, einzigartig, unglaublich

JA: erheblich, bedeutsam, relevant
JA: innovativ, neuartig
JA: bemerkenswert, substanziell
```

## Absolutismen

```
NEIN: immer, nie, alle, keiner
NEIN: zweifellos, offensichtlich, selbstverständlich

JA: häufig, selten, die meisten, wenige
JA: vermutlich, wahrscheinlich
JA: in der Regel, tendenziell
```

## Gedankenstriche (sparsam verwenden)

```
NEIN: das begrenzte Kontextfenster – also die begrenzte Menge – stellt...
JA:   das begrenzte Kontextfenster, also die begrenzte Menge, stellt...

NEIN: Content-Generierung und -Optimierung
JA:   Content-Generierung und Content-Optimierung
```

## Eingeschobene Nebensätze

Lange Einschübe zwischen Kommas unterbrechen den Lesefluss:
```
NEIN: Die identifizierten Probleme, fragmentiertes Vorgehen, Kontextverlust
      und mangelnde Strukturierung, verweisen auf einen Handlungsbedarf.

JA:   Die identifizierten Probleme verweisen auf einen Handlungsbedarf:
      fragmentiertes Vorgehen, Kontextverlust und mangelnde Strukturierung.
```

## Wiederholungen

Gleiche Wörter nicht in benachbarten Sätzen wiederholen:
```
NEIN: "...sowie vier Unterfragen ab... werden vier Unterfragen adressiert"
JA:   "...sowie mehrere Unterfragen ab... werden vier Unterfragen adressiert"
```

## Füllwörter und Veraltetes

```
NEIN: im Prinzip, grundsätzlich (wenn überflüssig)
NEIN: sodann, alsdann, hernach (veraltet)
NEIN: Gang der Untersuchung (gestelzt)
```

## Explizite Kapitelverweise

```
NEIN: Die in Kapitel 2.2 dargestellten Konzepte...
NEIN: Wie im vorangegangenen Abschnitt erläutert...
NEIN: Im folgenden Kapitel wird dargestellt...

JA: Konzeptnamen direkt verwenden (implizite Verbindung)
```
