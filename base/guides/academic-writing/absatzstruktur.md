# Absatzstruktur (MEAL-Prinzip)

## Das MEAL-Prinzip

Jeder Absatz folgt dieser Struktur:

1. **M**ain Point -- Kernaussage des Absatzes (1 Satz)
2. **E**vidence -- Beleg durch Quelle (1-2 Sätze)
3. **A**nalysis -- Eigene Einordnung/Interpretation (1-2 Sätze)
4. **L**ink -- Inhaltliche Verbindung zum nächsten Gedanken (1 Satz)

## Absatz-Richtwerte

- **Länge:** 80-150 Wörter pro Absatz (4-6 Sätze)
- **Pro Seite:** 4-7 Absätze (je nach Absatzlänge und Zeilenabstand)
- **Minimum:** 2 Absätze pro Unterkapitel

## Absatztypen

| Typ | Zweck | Wörter | Zitate |
|-----|-------|---------|--------|
| Definition | Begriff einführen/abgrenzen | 80-120 | 1-2 |
| Argument | These mit Beleg stützen | 100-150 | 1-2 |
| Analyse | Erkenntnisse einordnen/bewerten | 100-150 | 0-1 |
| Vergleich | Perspektiven gegenüberstellen | 100-150 | 2 |
| Überleitung | Gedanken verbinden | 60-100 | 0 |

## Häufige Fehler

### Zu lange Absätze
Ein Absatz der mehr als einen Hauptgedanken enthält sollte aufgeteilt werden.

### Absätze ohne Kernaussage
Jeder Absatz braucht EINEN klaren Hauptgedanken im ersten Satz.

### Fehlende Analyse
Nicht nur Quellen aneinanderreihen -- eigene Einordnung ist Pflicht.

### Redundanz
```
NEIN: Die KI-Technologie ist eine Technologie, die...
JA:   KI-Technologie ermöglicht...
```

### Schwache Verben
```
NEIN: Es gibt viele Unternehmen, die...
JA:   Zahlreiche Unternehmen setzen...

NEIN: Man kann sagen, dass...
JA:   Es zeigt sich, dass...
```

### Übertriebene Nominalisierungen
```
NEIN: Die Durchführung einer Implementierung...
JA:   Die Implementierung...
```
