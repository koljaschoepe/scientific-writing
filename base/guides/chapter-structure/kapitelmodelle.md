# Kapitelmodelle

## Seminararbeit-Modell (10-20 Seiten)

### Variante A: Mit Quellen
```
1. Einleitung
   1.1 Problemstellung
   1.2 Zielsetzung
2. Hauptteil
   2.1 [Thematischer Schwerpunkt 1]
   2.2 [Thematischer Schwerpunkt 2]
   2.3 [Thematischer Schwerpunkt 3] (optional)
3. Fazit
```

### Variante B: Ohne Quellen
```
1. Einleitung
   1.1 Problemstellung und Zielsetzung
2. Hauptteil
   2.1 [Thematischer Schwerpunkt 1]
   2.2 [Thematischer Schwerpunkt 2]
3. Fazit
```

## Hausarbeit-Modell (15-25 Seiten)

```
1. Einleitung
   1.1 Problemstellung
   1.2 Zielsetzung und Aufbau
2. Grundlagen
   2.1 [Zentraler Begriff / Theorie]
   2.2 [Weiterer Grundlagenbegriff]
3. Hauptteil
   3.1 [Analyse / Argumentation 1]
   3.2 [Analyse / Argumentation 2]
4. Fazit
```

## 3-Kapitel-Modell (Kurzarbeiten)

Für Arbeiten bis ca. 30 Seiten.

```
1. Einleitung
2. [Hauptteil: Theorie + Analyse kombiniert]
3. Fazit
```

## 4-Kapitel-Modell (Standard kompakt)

Für Arbeiten von 30-45 Seiten.

```
1. Einleitung
2. Theoretische Grundlagen
3. [Hauptteil: Analyse/Konzeption]
4. Fazit und Ausblick
```

## 5-Kapitel-Modell (Standard)

Das verbreitetste Modell für Bachelorarbeiten (40-60 Seiten).

```
1. Einleitung
   1.1 Problemstellung und Relevanz
   1.2 Zielsetzung und Forschungsfrage
   1.3 Aufbau der Arbeit

2. Theoretische Grundlagen
   2.1 [Zentraler Begriff 1]
   2.2 [Zentraler Begriff 2]
   2.3 [Theoretischer Rahmen/Modell]

3. [Hauptteil: Analyse/Empirie/Konzeption]
   3.1 [Aspekt 1]
   3.2 [Aspekt 2]
   3.3 [Aspekt 3]

4. [Handlungsempfehlungen/Diskussion]
   4.1 [Empfehlung/Ergebnis 1]
   4.2 [Empfehlung/Ergebnis 2]
   4.3 [Empfehlung/Ergebnis 3]

5. Fazit und Ausblick
   5.1 Zusammenfassung der Erkenntnisse
   5.2 Kritische Würdigung und Limitationen
   5.3 Ausblick
```

## 6-Kapitel-Modell (Erweitert)

Für umfangreichere Arbeiten (50-80 Seiten) oder Masterarbeiten.

```
1. Einleitung
2. Theoretische Grundlagen
3. [Analyse Teil 1]
4. [Analyse Teil 2 / Empirie]
5. Diskussion und Handlungsempfehlungen
6. Fazit und Ausblick
```

## Empirische Variante (5-Kapitel)

Für Arbeiten mit eigener Datenerhebung (Umfrage, Interview, Experiment).

```
1. Einleitung
2. Theoretische Grundlagen und Forschungsstand
3. Methodisches Vorgehen
   3.1 Forschungsdesign
   3.2 Datenerhebung
   3.3 Datenauswertung
4. Ergebnisse und Diskussion
   4.1 Darstellung der Ergebnisse
   4.2 Interpretation
   4.3 Einordnung in den Forschungsstand
5. Fazit und Ausblick
```

## Empirische Variante (6-Kapitel)

Methodik und Ergebnisse getrennt.

```
1. Einleitung
2. Theoretische Grundlagen
3. Methodisches Vorgehen
4. Ergebnisse
5. Diskussion und Handlungsempfehlungen
6. Fazit und Ausblick
```

## Modell wählen

| Arbeitstyp | Seiten | Empfohlenes Modell |
|------------|--------|--------------------|
| Seminararbeit (mit Quellen) | 10-20 | Seminararbeit Variante A |
| Seminararbeit (ohne Quellen) | 10-20 | Seminararbeit Variante B |
| Hausarbeit | 15-25 | Hausarbeit-Modell |
| Bachelorarbeit (kompakt) | 40-50 | 4-Kapitel |
| Bachelorarbeit (Standard) | 50-60 | 5-Kapitel |
| Masterarbeit | 50-100 | 5- oder 6-Kapitel |
| Dissertation | 150+ | 6+-Kapitel |
| Eigene Datenerhebung | beliebig | Empirische Variante |

## Kapitelspezifische Anforderungen

### Einleitung
- Vom Allgemeinen zum Spezifischen
- Relevanz (gesellschaftlich, wirtschaftlich, wissenschaftlich)
- Präzise, beantwortbare Forschungsfrage(n)
- Aufbau der Arbeit (KEINE Verbalisierung des Inhaltsverzeichnisses)

### Theoretische Grundlagen
- Nur das aufnehmen, was für die Arbeit relevant ist
- Begriffe definieren und abgrenzen
- Stand der Forschung darstellen

### Hauptteil
- Größter und wichtigster Teil
- Unterkapitel ausgewogen gewichten
- Kernanalyse zur Beantwortung der Forschungsfrage

### Handlungsempfehlungen/Diskussion
- Empfehlungen müssen aus der Analyse hervorgehen
- Konkret und umsetzbar formulieren
- Praxisbezug herstellen

### Fazit
- KEINE neuen Fakten oder Quellen
- Beantwortung der Forschungsfrage
- Ehrliche Reflexion über Limitationen

### Abstract / Zusammenfassung (optional, je nach Hochschule)
- Steht VOR der Einleitung (nach Inhaltsverzeichnis, vor Kapitel 1)
- 150-300 Wörter (halbe bis ganze Seite)
- Enthält: Thema, Fragestellung, Methodik, zentrale Ergebnisse, Schlussfolgerung
- Wird als LETZTES geschrieben (nach allen Kapiteln, vor der Finalisierung)
- Keine Zitationen im Abstract
- Eigenständig verständlich (ohne Rest der Arbeit lesen zu müssen)
- Bei `config.verzeichnisse.abstract: true` wird ein Abstract generiert
