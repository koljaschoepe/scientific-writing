# Kapitelgewichtung

## Grundformel

Die Gewichtung basiert auf dem Gesamtumfang des Textteils (ohne Verzeichnisse).

| Kapiteltyp | Anteil | Formel |
|------------|--------|--------|
| Einleitung | 8-10% | Seitenzahl × 0.08-0.10 |
| Theoretische Grundlagen | 20-25% | Seitenzahl × 0.20-0.25 |
| Hauptteil (Analyse/Empirie) | 35-45% | Seitenzahl × 0.35-0.45 |
| Handlungsempfehlungen/Diskussion | 15-25% | Seitenzahl × 0.15-0.25 |
| Fazit und Ausblick | 8-10% | Seitenzahl × 0.08-0.10 |

## Symmetrieregel

- Einleitung ≈ Fazit (gleicher Umfang)
- Grundlagen ≈ Empfehlungen/Diskussion
- Hauptteil = größter Teil (mindestens ein Drittel)

## Anpassung nach Arbeitstyp

**Literaturarbeit:**
- Grundlagen tendenziell größer (bis 25%)
- Hauptteil: systematische Analyse verschiedener Perspektiven

**Empirische Arbeit:**
- Hauptteil tendenziell größer (bis 45%)
- Methodik als eigener Abschnitt im Hauptteil

## Beispielrechnung

Bei 40 Seiten Textteil:

| Kapitel | Seiten |
|---------|--------|
| Einleitung | 3-4 |
| Grundlagen | 8-10 |
| Hauptteil | 14-18 |
| Empfehlungen | 6-10 |
| Fazit | 3-4 |

Bei 60 Seiten Textteil:

| Kapitel | Seiten |
|---------|--------|
| Einleitung | 5-6 |
| Grundlagen | 12-15 |
| Hauptteil | 21-27 |
| Empfehlungen | 9-15 |
| Fazit | 5-6 |

## Gewichtung: Seminararbeit (10-20 Seiten)

| Kapiteltyp | Anteil | Bei 15 Seiten |
|------------|--------|---------------|
| Einleitung | 10-12% | 1.5-2 |
| Hauptteil (2-3 Unterkapitel) | 75-80% | 11-12 |
| Fazit | 10-12% | 1.5-2 |

Beispiel bei 15 Seiten (Variante A, mit Quellen):

| Kapitel | Seiten |
|---------|--------|
| 1. Einleitung | 1.5-2 |
| 2.1 Schwerpunkt 1 | 3.5-4 |
| 2.2 Schwerpunkt 2 | 3.5-4 |
| 2.3 Schwerpunkt 3 | 3-3.5 |
| 3. Fazit | 1.5-2 |

## Gewichtung: Hausarbeit (15-25 Seiten)

| Kapiteltyp | Anteil | Bei 20 Seiten |
|------------|--------|---------------|
| Einleitung | 8-10% | 1.5-2 |
| Grundlagen | 20-25% | 4-5 |
| Hauptteil | 50-55% | 10-11 |
| Fazit | 8-10% | 1.5-2 |

Beispiel bei 20 Seiten:

| Kapitel | Seiten |
|---------|--------|
| 1.1 Problemstellung | 1 |
| 1.2 Zielsetzung und Aufbau | 1 |
| 2.1 Grundlagenbegriff 1 | 2-2.5 |
| 2.2 Grundlagenbegriff 2 | 2-2.5 |
| 3.1 Analyse 1 | 5-5.5 |
| 3.2 Analyse 2 | 5-5.5 |
| 4. Fazit | 1.5-2 |

## Warnsignale

- Ein Kapitel unter 5% des Gesamtumfangs: zu knapp, ggf. zusammenlegen
- Ein Kapitel über 50%: zu dominant, ggf. aufteilen
- Grundlagen länger als Hauptteil: falsche Gewichtung
