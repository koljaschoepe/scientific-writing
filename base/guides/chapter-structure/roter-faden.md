# Roter Faden

## Kernprinzip

Jedes Kapitel trägt zur Beantwortung der Forschungsfrage bei.
Der Leser muss jederzeit nachvollziehen können, WARUM ein Abschnitt an dieser Stelle steht.

## Prüfkriterien pro Unterkapitel

Für JEDES Unterkapitel vor und nach dem Schreiben prüfen:

1. **Relevanz:** Trägt dieses Kapitel zur Beantwortung der Forschungsfrage bei?
2. **Logik:** Baut es auf dem vorherigen Kapitel auf?
3. **Vorbereitung:** Bereitet es das nächste Kapitel inhaltlich vor?
4. **Abgrenzung:** Gibt es Überschneidungen mit anderen Kapiteln?
5. **Reihenfolge:** Werden Inhalte vorweggenommen, die erst später kommen?

## Verbotene Muster

- Wiederholungen: Gleicher Inhalt in verschiedenen Kapiteln
- Vorwegnahmen: Ergebnisse bereits in den Grundlagen
- Thematische Sprünge: Abrupter Themenwechsel ohne logische Verbindung
- Isolierte Kapitel: Kein erkennbarer Bezug zum Rest der Arbeit

## Kohärenz-Werkzeuge

### Konzeptverknüpfung
Zentrale Begriffe, die in einem Kapitel eingeführt werden, tauchen in späteren
Kapiteln wieder auf. Das schafft Verbindung ohne explizite Verweise.

### Forschungsfrage als Anker
Die Forschungsfrage dient als Orientierung. In der Einleitung formuliert,
trägt jedes Kapitel einen Aspekt zur Beantwortung bei. Im Fazit wird sie
explizit beantwortet.

### Argumentationskette
```
Grundlagen definieren Begriffe
  -> Analyse wendet Begriffe an
    -> Empfehlungen leiten sich aus der Analyse ab
      -> Fazit beantwortet die Forschungsfrage
```

## Planungs-Checkliste (vor dem Schreiben)

Für jedes Unterkapitel klären:

- [ ] Kernaussage des Kapitels?
- [ ] Welche Quellen werden verwendet?
- [ ] Welche Argumente werden aufgebaut?
- [ ] Was gehört NICHT in dieses Kapitel?
- [ ] Welche Begriffe aus vorherigen Kapiteln werden aufgegriffen?

## Review-Checkliste (nach dem Schreiben)

- [ ] Forschungsfrage wird durch das Kapitel bedient?
- [ ] Keine Inhalte aus anderen Kapiteln wiederholt?
- [ ] Keine Ergebnisse vorweggenommen?
- [ ] Übergänge folgen den Regeln aus `uebergaenge.md`?
- [ ] Begriffe konsistent mit den Definitionen aus den Grundlagen?
